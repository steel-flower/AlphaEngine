## 🎯 배포 3단계 (약 10분 소요)

### 1단계: GitHub 저장소 생성 (3분)

1. **GitHub 접속**: https://github.com
   - 로그인 (계정 없으면 무료 가입)

2. **새 저장소 생성**:
   - 우측 상단 `+` 버튼 → `New repository` 클릭
   - **Repository name**: `alpha-engine-private` (원하는 이름)
   - **Visibility**: ⭐ **Private** 선택 (중요!)
   - **Initialize this repository with**: 체크 안 함
   - `Create repository` 클릭

3. **저장소 URL 복사**:
   - 생성된 페이지에서 HTTPS URL 복사
   - 예: `https://github.com/YOUR_USERNAME/alpha-engine-private.git`

---

### 2단계: 코드 업로드 (5분)

#### 방법 A: GitHub Desktop 사용 (초보자 추천 ⭐)

1. **GitHub Desktop 설치**:
   - https://desktop.github.com 에서 다운로드 및 설치
   - GitHub 계정으로 로그인

2. **프로젝트 추가**:
   - `File` → `Add local repository`
   - 폴더 선택: `C:\Users\user\Desktop\Antigravity\AlphaEngine`
   - "create a repository" 또는 "initialize" 클릭

3. **업로드**:
   - 좌측 하단 Summary: `Initial commit - Alpha Engine v7.7`
   - `Commit to main` 클릭
   - 상단 `Publish repository` 클릭
   - ⭐ **Keep this code private** 체크
   - `Publish repository` 클릭

#### 방법 B: Git 명령어 사용 (고급 사용자)

PowerShell에서 실행:

```powershell
cd C:\Users\user\Desktop\Antigravity\AlphaEngine

# Git 초기화
git init

# 파일 추가
git add .

# 커밋
git commit -m "Initial commit - Alpha Engine v7.7"

# GitHub 연결 (YOUR_USERNAME을 본인 아이디로 변경!)
git remote add origin https://github.com/YOUR_USERNAME/alpha-engine-private.git

# 업로드
git branch -M main
git push -u origin main
```

---

### 3단계: Streamlit Cloud 배포 (2분)

1. **Streamlit Cloud 접속**:
   - https://share.streamlit.io
   - `Sign in with GitHub` 클릭
   - GitHub 계정으로 로그인

2. **앱 배포**:
   - `New app` 버튼 클릭
   - 설정:
     - **Repository**: `YOUR_USERNAME/alpha-engine-private` 선택
     - **Branch**: `main`
     - **Main file path**: `app.py`
   - `Deploy!` 클릭

3. **배포 대기**:
   - 약 2-5분 소요
   - 화면에서 진행 상황 확인
   - ✅ "Your app is live!" 메시지 확인

## 🔐 비밀번호 설정 (필수!)

배포 완료 후:

1. **앱 설정 열기**:
   - 배포된 앱 페이지 우측 상단 `⋮` (메뉴) 클릭
   - `Settings` 선택

2. **Secrets 추가**:
   - 좌측 메뉴 `Secrets` 선택
   - 다음 내용 입력:

```toml
password = "your_strong_password_2026"
```

   - `your_strong_password_2026`을 원하는 비밀번호로 변경
   - `Save` 클릭

3. **자동 재시작**:
   - 앱이 자동으로 재시작됨
   - 새 비밀번호가 적용됨

## 🆘 문제 해결

### 배포 실패 시
1. Streamlit Cloud 로그 확인
2. `requirements.txt` 패키지 확인
3. GitHub 저장소가 Public이 아닌 Private인지 확인

### 느린 속도
- 정상입니다! 무료 플랜의 제한
- 캐시가 작동하면 빨라짐
- 새로고침 버튼으로 캐시 초기화 가능

### 비밀번호 분실
1. Streamlit Cloud → Settings → Secrets
2. 새 비밀번호로 변경
3. 저장 후 재시작
