## 🚀 다음 단계 (1분!)

### 1. Gmail 앱 비밀번호 발급
```
EMAIL_설정가이드.md 파일을 열어서 따라하세요!

주요 단계:
1. https://myaccount.google.com/apppasswords 접속
2. 앱 비밀번호 생성 (16자리)
3. 복사
```

### 2. 설정 파일 작성
```
email_config.json 파일에 입력:
- sender_email: 본인 Gmail
- sender_password: 16자리 앱 비밀번호
- receiver_email: 본인 Gmail
```

### 3. 테스트
```
run_email_test.bat 더블클릭
→ Gmail에서 테스트 메시지 확인
```

### 4. 실전 사용
```
run_monitor_v7.7.bat 더블클릭
→ 백그라운드 모니터링 시작
→ 신호 발생 시 이메일 알림
→ 카이로스 HTS에서 수동 주문
```
